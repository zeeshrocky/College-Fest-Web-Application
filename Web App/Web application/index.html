News

Department of Computer Science and Engineering at HKBK College of Engineering is organizing the Third State Level CSI Technical Symposium OVERLOAD++ on 5th April 2015.

This is a launch platform for students to showcase and test their creativity, programming, knowledge and communication skills. The parallel programming contest is the highlight of this event. It tests the students in their proficiency to parallelize a given sequential code in C using OpenMP / CUDA for execution on the given HPC architecture.
